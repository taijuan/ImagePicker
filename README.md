# ImagePikcer[![](https://www.jitpack.io/v/taijuan/ImagePikcer.svg)](https://www.jitpack.io/#taijuan/ImagePikcer)

## 集成
#### 1. 添加maven:
    allprojects {
	    repositories {
	        maven { url 'https://www.jitpack.io' }
	    }
	}
#### 2. module添加引用
	dependencies {
	    implementation 'com.github.taijuan:ImagePikcer:1.0.0'
	}
