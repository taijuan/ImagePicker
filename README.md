# androidJitPack[![](https://jitpack.io/v/taijuan/ImagePicker.svg)](https://jitpack.io/#taijuan/ImagePicker)

## 集成
#### 1. 添加maven:
    allprojects {
	    repositories {
	        maven { url 'https://www.jitpack.io' }
	    }
	}
#### 2. module添加引用
	dependencies {
	    图片视频选择
        implementation 'com.github.taijuan.androidJitpack:picker:1.1.0'
	}
